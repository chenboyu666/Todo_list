# Todo list

`Todo list` 是一款 Windows 桌面待办与时间管理工具。它可以悬浮在桌面上，帮助你持续关注当前任务、工作计时、截止倒计时、提醒状态和历史完成情况。

## 下载与运行

在 Releases 页面下载最新版：

[下载 Todo-list-V1.0-windows.exe](https://github.com/chenboyu666/Todo_list/releases/download/v1.0/Todo-list-V1.0-windows.exe)

下载后双击 `Todo-list-V1.0-windows.exe` 即可运行。首次启动时，程序会在同级目录自动创建 `data` 文件夹，用来保存任务、设置、历史记录、背景图片和自定义图标等数据。

## 主要功能

- 桌面悬浮窗口：无边框界面，支持拖动、缩放、置顶、托盘恢复和鼠标穿透。
- 实时时间面板：显示数字时钟，并统计今日完成、进行中、临近截止和超时任务。
- 多任务管理：支持新增、编辑、删除、完成、暂停、继续、置顶当前任务。
- 优先级区分：任务可设置高、中、低优先级，并通过图标、颜色和卡片状态区分。
- 工作计时与截止倒计时：同时展示实际工作计时和截止剩余时间。
- 任务提醒：临近截止、超时和完成任务时会出现提醒小窗。
- 背景与外观设置：支持透明度、界面缩放、背景图片、随机背景文件夹和程序图标设置。
- 历史任务：查看已完成任务，可按日期、优先级、状态、标签和任务名称筛选。
- 历史统计：查看优先级结构、完成趋势、准时/超时分布和标签统计。
- 3D 洞察图：根据已完成任务的标题、标签、备注和体会提取关键词，生成类似 Obsidian 的任务关系图。
- CSV 导出：可按日期范围导出历史任务数据。

## 基础使用

1. 双击 `Todo-list-V1.0-windows.exe` 启动程序。
2. 点击新增任务，填写任务名称、优先级、预估工作量、截止时间和备注。
3. 在任务列表中选择任务，可将其设为当前任务。
4. 当前任务区域会显示工作计时、截止倒计时、完成/删除等常用操作。
5. 点击历史任务，可查看完成记录、统计图表、3D 洞察图和 CSV 导出。
6. 点击设置，可调整提醒、外观、背景、图标、托盘和窗口行为。

## 3D 洞察图怎么用

历史窗口中的“洞察”页面会把已完成任务变成关系图：

- 任务节点代表已完成任务。
- 关键词节点来自任务名称、标签、备注和完成体会。
- 节点之间的连线表示这些任务存在共同关键词。
- 拖动画面可以旋转图谱，滚轮可以缩放。
- 点击关键词节点，可以查看相关任务。
- 点击任务节点后，可以补备注、改标签或导出历史数据。

这个页面适合用来发现重复主题、关联任务、孤立记录和需要补充标签的任务。

## 数据保存

程序会在 exe 同级目录创建：

```text
data\
```

这里保存任务、设置、历史记录和自定义资源。移动程序时，请把 `Todo-list-V1.0-windows.exe` 和同级 `data` 文件夹一起移动。

## 注意事项

- 程序适用于 Windows 系统。
- 如果系统提示未知发布者，请确认文件来自本项目 Releases 页面后再运行。
- 关闭窗口后，程序可能会按设置最小化到托盘；可从托盘图标恢复或退出。
